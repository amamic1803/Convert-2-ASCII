## Downloading
Prebuilt program is available under Releases

## Building
1. Clone repository
2. Install Python
3. Install dependencies: `pip install -r requirements.txt`
4. Install UPX (optional)
5. Run build.py
6. Built `*.exe` will be placed in the same folder
